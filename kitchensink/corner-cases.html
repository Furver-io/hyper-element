Direct Instantiation Blocked

Attempting to call new hyperElement() directly should throw an error. The Proxy construct trap blocks direct instantiation while allowing subclass extension.

Nested Hole Template Switch (getHole else branch)

Tests the getHole else branch in hole.js where a nested Hole switches to a different template. This triggers hole.n.replaceWith(dom(value)) when hole.t !== value.t.

Event Array Same Handler Skip (hole.js EVENT optimization)

Tests the else if (type & EVENT && value[0] === prev[0]) branch in hole.js update(). When an event is passed as @click=${[handler, options]} and the handler reference doesn't change between renders, the update is skipped (continue).

Boolean Attribute: HTML vs XML Mode (Element.toString)

Tests the boolean attribute handling in Element.toString() from nodes.js. When typeof value === 'boolean' is true:

  • HTML mode: outputs ` ${key}` (bare attribute)
  • XML/SVG mode: outputs ` ${key}=""` (empty value)
Uses STATIC boolean attributes (no value) to trigger this code path.

PersistentFragment Diff Operations (diffFragment -1 and -0)

Tests the diffFragment function's handling of multi-node fragments during array operations. Specifically targets:

  • drop(node) - triggered by array item REMOVAL (operation -1)
  • node.lastChild - triggered by array REORDERING (operation -0)
These branches only execute for PersistentFragments (nodeType === 11) during the diff algorithm's reconciliation of array-based templates.

dom() Re-render Same Hole

Tests calling dom(hole) twice on the same Hole instance. The second call should hit the hole.n truthy branch in hole.js:16, calling hole.update(hole).
