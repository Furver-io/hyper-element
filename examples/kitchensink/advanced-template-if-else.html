If/Else Conditional

{#if condition}...{else}...{/if} should render the else branch when condition is falsy. 

customElements.define(
      'if-else-elem',
      class extends hyperElement {
        render(Html) {
          Html`${Html.template({ active: false })}`;
        }
      }
    );
{+if active}ACTIVE{else}INACTIVE{-if}
