Content Handling

Demonstrates accessing content placed between element tags via this.wrappedContent.

Wrapped Content

The this.wrappedContent property contains the original innerHTML.

customElements.define(
  'content-element',
  class extends hyperElement {
    render(Html) {
      // Transform the wrapped content
      Html`Transformed: ${this.wrappedContent.trim().toUpperCase()}`;
    }
  }
);
