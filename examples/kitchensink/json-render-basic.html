JSON Render Basic

Demonstrates the simplest use of the json-render module: rendering a JSON spec declaratively using the <jr-ui> custom element without writing JS.

Declarative JSON Rendering

Render a spec simply by attaching a JSON payload to the data-spec attribute.

<jr-ui id="target-jr-ui" data-spec='{"root":"msg","elements":{"msg":{"type":"Text","props":{"content":"Rendered from pure JSON spec!"}}}}'></jr-ui>
