JSON Render Basic

Demonstrates the simplest use of the json-render module: rendering a JSON spec declaratively using the <json-render> custom element without writing JS.

Declarative JSON Rendering

Render a spec by placing the JSON payload as the body text content of the element — no attribute, no escaping.

<json-render id="target-json-render">{"root":"msg","elements":{"msg":{"type":"Text","props":{"content":"Rendered from pure JSON spec!"}}}}</json-render>
