Basic Rendering

Demonstrates the simplest use of hyper-element: defining a custom element with a render method.

Simple Element

A minimal custom element that renders static content.

customElements.define(
  'simple-element',
  class extends hyperElement {
    render(Html) {
      Html`Hello from hyper-element!`;
    }
  }
);
document.getElementById('simple-output').innerHTML = '<simple-element></simple-element>';
