Same-Type Block Nesting - Depth Tracking Coverage

Tests to exercise findMatchingClose depth tracking (lines 70-73 of parseEachBlocks.js). These tests trigger the depth tracking code path even when nested same-type blocks with interpolation aren't fully processed.

1. Depth Tracking for Nested {+each}

The outer loop should process correctly. The depth tracking skips the inner {+each} when finding the outer's {-each}.

customElements.define(
  'depth-track-each',
  class extends hyperElement {
    render(Html) {
      const outer = ['A', 'B'];
      const inner = ['x'];
      Html`<div id="result-1">{+each ${outer}}<span class="outer">{...}</span><span class="nested">{+each ${inner}}{...}{-each}</span>{-each}</div>`;
    }
  }
);

const elem = document.createElement('depth-track-each');
document.getElementById('output-1').appendChild(elem);
