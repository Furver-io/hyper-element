Attributes

Demonstrates accessing element attributes via this.attrs object.

Reading Attributes

The this.attrs object provides access to all element attributes.

customElements.define(
  'attrs-element',
  class extends hyperElement {
    render(Html) {
      Html`Greeting: ${this.attrs.greeting}`;
    }
  }
);

const attrsElem = document.createElement('attrs-element');
attrsElem.setAttribute('greeting', 'Hello World');
document.getElementById('attrs-output').appendChild(attrsElem);
